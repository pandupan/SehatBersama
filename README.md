# 🏥 SehatBersama

Aplikasi **layanan kesehatan bersama** yang memudahkan akses informasi kesehatan bagi masyarakat. Dibangun dengan Next.js untuk pengalaman pengguna yang optimal.

## ✨ Fitur (Rencana/Pengembangan)

- Informasi layanan kesehatan
- Konsultasi online
- Manajemen jadwal
- Artikel kesehatan

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **shadcn/ui** — UI components

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
